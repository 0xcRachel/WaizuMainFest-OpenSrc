## Usage Flow

### 1. Search for a game
Type the game name into the search box and press **Enter**. Select a game from the dropdown suggestions.

### 2. View game information
After selecting a game, the app automatically displays:
- Game cover image
- Game name & App ID
- Developer & Release date
- Minimum system requirements

### 3. Check Manifest status
The app automatically checks if a manifest exists for the selected game.

**Status will show:**
- ✅ Available — ready to download
- ❌ Not found — displays list of available App IDs

### 4. Download Manifest
If manifest is available, click the **Download** button to get the manifest file.

### 5. Use with SteamTools
After downloading the manifest, use **SteamTools** to add or unlock the game on your Steam account.

## Interface

| Section | Description |

- Windows 10 / 11
